## Project Configuration

- **Language**: TypeScript
- **Package Manager**: npm
- **Add-ons**: none

---

PocketRugbyNL is a lightweight sveltekit progressive webapp to quickly
find matches for from the NL rugby competition. Users can filter
teams/clubs (multiselect) to view all relevant matches. It remembers
the last choice when you open it again.

https://rugby.nl/competitie/speelschema/
